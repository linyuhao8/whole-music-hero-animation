# Whole Music Hero

This is a web design animation project created for my client’s music and event brand, Whole Music Studio.

I helped design a responsive hero section with scroll-based motion effects, an animated visual background, marquee typography, and a modern portfolio-style landing page experience to strengthen the brand’s digital visual identity.
